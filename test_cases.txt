# 테스트 케이스 1: 기본 예제 (N=3, K=7)
입력:
3 7

예상 출력: 6
설명: 3x3 배열 [1,2,3,2,4,6,3,6,9]를 정렬하면 [1,2,2,3,3,4,6,6,9]
       7번째 수는 6

# 테스트 케이스 2: 작은 값 (N=3, K=1)
입력:
3 1

예상 출력: 1
설명: 첫 번째 수는 항상 1

# 테스트 케이스 3: 최대값 (N=3, K=9)
입력:
3 9

예상 출력: 9
설명: 마지막 수는 N*N

# 테스트 케이스 4: 중간값 (N=4, K=8)
입력:
4 8

예상 출력: 4
설명: 4x4 배열을 정렬했을 때 8번째 수

# 테스트 케이스 5: 큰 N (N=100000, K=1)
입력:
100000 1

예상 출력: 1
설명: 경계값 테스트

# 테스트 케이스 6: 큰 N과 중간 K (N=100000, K=50000)

100000 50000
